Getting Started

To get started on using the library, please include the view.js script file, as well as the style.css in your html file. Also, since the library utilizes external library 'jQuery', please also include jQuery in your html file as well.(Note: the current link for including jQuery is following: https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js).

Below are some implementation of several basic functionalities. For more details of all the functionalities, please see the API section below.

Adding sections:

user adds the sections to display in the list to fill the panels.

Sliding side panel:

user can drag to widen or narrow the width of the side panel.

Drag and drop:

user can drag and drop sections to rearrange the order.

Swap between format/layout:

user can change the layout of the panel through clicking the button.

API

Below are the details of all the objects and functions in the library.


Objects

  • createToggleButton()

    To create the toggler button appeared on the top left of the page, used to show or hide the side panel.

  • createSliderButton()

    To create the grey vertical bar appeared on the side of the panel, used to dynamically change the width of the side panel.

  • navAndSideExchange()

    To create the SWAP button to interchange between horizontal and vertical layout.



Functions

  • addSection(sectionName)

    To insert a new section in the section list of the side panel, the name will be specified by the argument "sectionName".

  • removeSection(sectionName)

    To delete a previously defined section from the section list of the side panel, the section to be deleted is specified by the argument "sectionName".

  • addNestedSections(parentSectionName, nestedSubs)

    To insert a nested list under a previously defined section from the section list of the side panel. The section the nested list to be inserted, is specified by the argument parentSectionName, and the nested list is specified by nestedSubs, which is an array of section names to insert.

  • showHideNested()

    To hide or show the nested list inserted. This function enables the nested list to be displayed, otherwise the nested list is set as "displayed = 'none'" by default.

  • sliding()

    To achieve the dragging functionality for the side panel, it dynamically monitors the mouse position upon clicked, and adjust the width of the side panel according to the current mouse position with respect to the position when the slider is clicked.

  • dragAndDrop()

    To enable the functionality of the drag and drop to reorder the section list. The sections are set to have attribute of draggable to be true, and this function monitors which section is being dragged through event listener, and determines from mouse position to determine where its dropped location.

  • toNav()

    To enable the swap between horizontal and vertical format, this function specifies the style and all the settings and specifications for the horizontal layout, for all of the objects.

  • toSide()

    To enable the swap between horizontal and vertical format, this function specifies the style and all the settings and specifications for the vertical layout, for all of the objects.

  • navAndSideExchange()

    With the previous two functions, namely toNav() and toSide(), this function creates the "SWAP" button and uses if statement to determine the current layout mode, and swaps/toggles to the other one.
